Raytracing Tester

Compare the performances of different raytracing approaches:
- without workers,
- with N workers working in parallel.

Depending on the model, the number of workers, the number of ray traced,
and the hardware, the results may vary.
While using workers, the three js scene is duplicated N times in each workers.
To do so:
- first the scene have to be converted to json before being passed to the workers.
- then the workers have to convert back the json scene to three js scene.

Check the console for computation time.
